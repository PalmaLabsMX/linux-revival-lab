# linux-revival-lab
Reviviendo hardware antiguo usando Linux Mint Xfce 💚🐧

En este primer repositorio quisiera compartirles mi reciente experiencia usando una de las distros de Linux más famosas y recomendadas por su simplicidad técnica. 

Como lo vieron en la descripción, se trata nada más y nada menos que de **Linux Mint**. 
Si bien actualmente están disponibles 3 sabores o ediciones de Linux Mint en la página oficial, elegí Xfce debido a que es la más ligera y eficiente para con los recursos de cómputo de modelos antiguos. 

En este caso he logrado rescatar dos laptops que estaban a punto de ser cruelmente destruidas y deshechadas, pues llevaban años sin usarse y según sus usuarios, eran extremadamente lentas, además de que una de ellas es tan vieja que aún conserva puertos ethernet y seriales.
Al encenderlas me percaté de que ambas dieron imagen sin problemas y las baterías aún vivían. Luego confirmé que en efecto, estaban demasiado lentas incluso para su época. 

El culpable principal: Windows 

>[!Note]
>*Cabe aclarar que este repositorio no es una crítica directa al sistema operativo Windows, ni mucho menos una invitación a dejar Windows sin razón. De hecho mi objetivo es poner en perspectiva ambos sistemas operativos y explicar por qué y cuándo es más conveniente usar Linux según tus necesidades...*

Tanto las nuevas versiones de Windows como algunas de las anteriores suelen ser muy pesadas y poco optimizadas, ya que no están pensadas para correr en hardware viejo o con recursos limitados.
